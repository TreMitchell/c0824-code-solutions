# fetch-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What does `fetch()` return?
  - `Fetch()` returns a promise that resolves to the Response object representing the outcome of the request.
- What is the default request method used by `fetch()`?
  - The default request method used by `fetch()` is GET.
- How do you specify the request method (`GET`, `POST`, etc.) when calling `fetch`?
  - To specify the request method for a fetch call, you must pass an options object as the second argument to fetch(), setting the method property to any of those methods.
- How does `fetch` report errors?
  - Fetch report errors by checking the response.ok property or the status code directly within the response handling logic.
