# html-intro-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- Where do you put non-visible content about the HTML document?
  - You would put non-visible content inside of the <head> element.
- Where do you put visible content about the HTML document?
  - You would put visible content about the HTML document inside of the <body> tag.
- Where do the `<head>` and `<body>` tags go in a valid HTML document?
  - The <head> and <body> tags would go inside the root element (<html>) in a valid HTML document.
- What is the purpose of a `<!DOCTYPE>` declaration?
  - To declare the document type and the version of html you are using.
- Give five examples of HTML element types.
  - <head>, <body>, <html>, <p>, <h1>
- What is the purpose of HTML attributes?
  - HTML attributes contain any info and/or functionality for elements, usually in name/value pairs.
- Give an example of an HTML entity (escape character).
  - An example of an HTML entity is (&quot;).
